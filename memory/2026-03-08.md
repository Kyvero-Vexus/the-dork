# Search Log - 2026-03-08

## LML (Lisp Machines Lisp) Investigation

### Task
Find information about Lisp Machines Lisp (LML) - the dialect used on Lisp Machines Inc. (LMI) machines. LMI was a Symbolics competitor in the 1980s.

### Key Findings

#### Terminology Clarification
- **Lisp Machine Lisp (LML)** is the general name for the dialect used on all MIT-derived Lisp machines
- **ZetaLisp** was Symbolics' name for their variant
- **LMI's variant** was called **ZETALISP-PLUS™** - this is what ran on LMI Lambda machines
- LMI and TI Explorer shared a common code base that differed from MIT/Symbolics

#### Historical Context
- LMI founded 1979 by Richard Greenblatt from MIT AI Lab
- Main product: LMI Lambda (and LMI-CADR before that)
- Went bankrupt 1987 before K-Machine could ship
- K-Machine: Original design (not CADR-descended), competitive with Cray-1 on Lisp benchmarks
- Detailed K-Machine architecture paper: http://fare.tunes.org/tmp/emergent/kmachine.htm

#### Key People
- David A. Moon, Richard Stallman, Daniel Weinreb - Original Lisp Machine Lisp designers
- Robert Powell, Bruce Deffenbaugh, Kent Hoult - K-Machine hardware designers
